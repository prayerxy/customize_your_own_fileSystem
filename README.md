### 项目的结构说明

- data_about_learning是准备阶段完成的学习记录
- 其余文件夹是项目结构
  - include是头文件夹
  - src是项目源文件夹
  - 外部makefile保证运行make
  - setup.sh是运行测试脚本

### 运行说明

- 方法1：直接运行脚本

  ```
  sudo bash setup.sh
  ```

- 方法2：也可以手动运行，先Make编译部署至linux内核，再挂载至一个文件夹下，对文件系统测试
